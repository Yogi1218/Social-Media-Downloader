# Social Media Downloader

Download videos from Instagram, X (Twitter), and YouTube Shorts 
directly from the terminal. Built with Python and yt-dlp, 
packaged as a standalone macOS app.

## Supported Platforms
- Instagram (Reels, posts)
- X / Twitter (videos)
- YouTube Shorts

> YouTube long-form videos are not supported due to DRM restrictions.

## Installation

### macOS (No setup needed)
Download `downloader.app` from Releases and open it.
ffmpeg is bundled — nothing else required.

### Run from source (any platform)
Requirements: Python 3.x, yt-dlp, ffmpeg

pip install yt-dlp
python3 main.py
